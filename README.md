# SCM Ops Skill

SCM (Source Control Management) 运维专家，专为老爹的外部大脑设计的 GitHub 与 GitLab 自动化管理协议。

## ✨ 特性
- **一键 GitHub 发版**: 自动处理 `package.json` 版本号、CHANGELOG 及 Git 推送。
- **GitLab 多开支持**: 完美适配官方 GitLab 与 Paylinker 内部实例。
- **Token 隔离管理**: 严格遵循安全 SOP，从不泄漏凭证。

## 🚀 安装
```bash
gemini skills install https://github.com/webkubor/scm-ops-skill
```

## 🛠 开发与调试
```bash
npm run validate
```
